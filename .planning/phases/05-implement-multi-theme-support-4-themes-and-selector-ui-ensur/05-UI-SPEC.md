## Color: Theme Palette Contract

The theme switcher will introduce 4 specific palettes mapped to CSS variables (`--bg-primary`, `--accent`, `--gradient-brand`, etc.)

1. **Midnight (Current Default)**
   - Dominant: `#070d1a` (Navy)
   - Secondary: `#0d1629`
   - Accent: `#0ea5e9` (Electric Blue)
   - Gradient: Navy / Electric Blue / Teal

2. **Forest (Dark)**
   - Dominant: `#05120c` (Deep Pine)
   - Secondary: `#0a1c15`
   - Accent: `#10b981` (Emerald)
   - Gradient: `#10b981` to `#34d399`

3. **Sunset (Dark)**
   - Dominant: `#170c10` (Dark Rose)
   - Secondary: `#211317`
   - Accent: `#f43f5e` (Rose)
   - Gradient: `#f43f5e` to `#f97316` (Orange)

4. **Snow (Light)**
   - Dominant: `#f8fafc` (Slate 50)
   - Secondary: `#ffffff`
   - Accent: `#2563eb` (Royal Blue)
   - Gradient: `#2563eb` to `#4f46e5`
   - Contrast adjustment: Requires inversion of text primary (`#0f172a`) and adjusting `--bg-card` for light glassmorphism (`rgba(0,0,0,0.03)`).

Accent reserved for: Active theme indicator, hover states, primary CTA.

---

## Interface: Theme Selector UI

- **Location**: Top navigation bar (right aligned) or floating bottom-right (depending on mobile constraints). We will place it inside the `#navbar > .nav-inner`.
- **Selector Type**: A glassmorphic dropdown or a horizontal pill with icons.
  - Using an icon button (e.g., Palette icon) that toggles a floating glassmorphic dropdown containing 4 theme buttons.
- **Micro-interactions**: 
  - Hovering a theme preview in the dropdown shows a subtle ring in its accent color.
  - Clicking sets `data-theme` on `<html>` and saves to `localStorage`.
  - CSS variables smoothly transition (`transition: background-color 0.4s ease`).
